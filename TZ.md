# Техническое задание: Цифровой книжный критик

## 1. Цель проекта
Сервис для анализа тональности и читабельности текстов.

## 2. Роли
- Гость: ввод текста и получение результатов
- Пользователь: сохранение истории анализов
- Администратор: доступ к Django Admin

## 3. Модели
- TextSample: образец текста (user, title, content, created_at)
- LexicalAnalysis: лексический анализ (1:1 с TextSample)
- SentimentScore: тональность по абзацам (M:1 с TextSample)

## 4. Ключевой функционал
- Анализ текста через Django Forms
- Расчёт индекса Флеша-Кинкейда
- Анализ тональности по абзацам
- Визуализация (Plotly)
- Облако слов с синонимами

## 5. Интеграции
- Яндекс.Словарь API
- Datamuse API
